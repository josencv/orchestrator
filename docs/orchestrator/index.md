## Interpretation Rules

- `Fact` means grounded in the provided source documents.
- `Decision` means introduced or tightened for this repo.
- `Provisional` means expected to be validated during implementation and may change without redefining the product.

## Design Posture

The orchestrator is Symphony-inspired, not Symphony-cloned.

The guiding rule for every doc in this folder is:

- preserve what Symphony got right,
- simplify what is too heavy for v0,
- add only what this project truly needs.
